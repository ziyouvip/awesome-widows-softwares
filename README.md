- 4.音乐播放歌词下载软件 MusicPlayer2

MusicPlayer2是一款集音乐播放、歌词显示、格式转换等众多功能于一身的音频播放软件。
支持歌词显示、歌词卡拉OK样式显示、歌词在线下载、歌词编辑、歌曲标签识别、专辑封面显示、专辑封面在线下载、频谱分析、音效设置、任务栏缩略图按钮、主题颜色、格式转换等功能。播放内核为BASS音频库(V2.4)。
播放器支持几乎所有常见的音频格式。
